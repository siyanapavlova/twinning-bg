# Twinning BG

Visualization of the twin towns of Bulgarian towns.

Inspired by [Map Men's video](https://www.youtube.com/watch?v=zte_vg0D8Z4) about [twin towns](https://en.wikipedia.org/wiki/Sister_city) in the UK.

The [data](https://www.namrb.org/uploads/tinymceup/files/TwinngBG2022Rev25082023.xls) comes from the [National Association of the Municipalities in the Republic of Bulgaria](https://www.namrb.org/en) and includes information about Bulgaria's municipalities and their twins as of 25th August 2023. For each entry, the original data includes, among others: name of the Bulgarian municipality, name of the twinned entity (in most cases town, village or municipality), country of the twinned entity, type of partnetship, partnership areas.

Location, Latin spelling, and Wikidata ID of each entity were added to the dataset as part of this project.

The original dataset includes more towns and relations than are displayed here (see below for exact numbers). A small portion of the difference in numbers comes from duplication. The rest is due to naming of twin entities that is too ambiguous (multiple towns of the same name in the same country) or unclear (spelling errors that don't allow for disamgbiguation).
